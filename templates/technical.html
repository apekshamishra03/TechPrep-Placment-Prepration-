💻 Practice Questions

1. Select All Records

Write a query to get all employees from 'employees' table.

2. Filter with WHERE

Get employees with salary > 50000.

SELECT * FROM employees WHERE salary > 50000;

3. Join Tables

Get employee names with their department names.

SELECT e.name, d.dept_name FROM employees e JOIN departments d ON e.dept_id = d.id;

4. Aggregate Functions

Find average salary by department.

SELECT dept_id, AVG(salary) FROM employees GROUP BY dept_id;
